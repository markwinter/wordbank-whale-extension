Exporting to Anki

Clicking the Export button will open a new tab with an Anki formatted text document. To import this file into Anki, first save the exported text file. Then in Anki, click the Import button and select the downloaded text file. You can then choose which fields in the text file correspond to which language in your card deck.

About

Word Bank was created with the aim to make it easier for language learners to save words they come across online for later learning. To quickly add new words, you can highlight the word and then right-click and select the Word Bank menu item.
